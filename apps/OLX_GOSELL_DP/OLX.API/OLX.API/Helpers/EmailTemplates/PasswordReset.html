Скидання пароля  

    Після натискання кнопки вам буде запропоновано виконати такі кроки:

  1. Введіть новий пароль.
  2. Підтвердьте новий пароль.
  3. Натисніть Надіслати.
Скинути пароль

Це посилання дійсне лише для одного використання. Термін дії закінчується через 2 години.

Якщо ви не надсилали запит на скидання свого паролю, проігноруйте це повідомлення або зв’яжіться з нашим відділом обслуговування клієнтів.
